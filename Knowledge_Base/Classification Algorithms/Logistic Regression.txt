Logistic Regression is fundamentally a classification algorithm, not a regression method.
Rather than predicting continuous numerical values, it predicts the probability that a given input belongs to a particular class. These probability outputs always fall between 0 and 1, making them perfect for binary decision-making tasks

Where Logistic Regression Shines:
- Spam Detection
- Disease Diagnosis
- Fraud Detection 
- Customer Churn

Logistic Regression is a model specifically designed for probability-based decisions-one that constraints outputs to valid probability ranges.

Classification as a Probability Problem:

The key insight behind logistic regression is reframing classification as probability estimation. Instead of directly predicting a class label, the model predicts the probability that an input belongs to a particular class.

- Model Predcition:
the algorithm calculates P(class = 1 | features), repressenting the probability that the input belongs to a class 1 given its feature values.
- Decision Threshold
a threshold (typically 0.5 converts the probability into a final classification. Probabilities above the threshold predict class 1; those below predict class 0.
- Final Classification
the model makes its decision: if P(spam) = 0.87, classify as spam. This threshold can be adjusted based on the relative costs of false positives versus false negatives.

The Sigmoid Function: The Heart of Logistic Regression
This mathematical transformation takes any real number as input and squashes it into the range (0, 1)-perfect for probabilioty interpretations.
This transformation is what allows logistic regression to maintain the simplicity of a linear model while producing valid probability outputs.

The architecture remains beautifully simple: logistic regression = linear model + sigmoid function. This elegtant combination gives us the interpretability of linear models with th probability outputs needed for classification.

How Logistic Regression Learns: Log Loss
Log loss heavily penalizes confident wrong predictions.
