Features strongly correlated with the target are prime candidates for driving predictions, while weakly correlated features may add noise or require transformation.

Code Example:

# Assuming 'price' is your target
target_corr = df.corr()["price"]
target_corr = target_corr.sort_values(ascending=False)
print(target_corr)

Use Cases
- Quick feature importance screening
- Baseline model feature selection
- Identifying transformation candidates
- Validating domain assumptions
